## 1. المقدمة والقيمة المقترحة

**محرك نمو واتساب الكويت** هو أشمل منصة مؤسسية لإدارة علاقات العملاء عبر واتساب، صُممت خصيصاً للسوق الكويتي والخليجي. على عكس المنافسين العالميين الذين يتعاملون مع الشرق الأوسط كسوق ثانوي، فإن كل مكون في هذه المنصة — من فهم اللهجة الكويتية بالذكاء الاصطناعي إلى معالجة مدفوعات كي-نت — صُمم من الأساس للكويت.

### لماذا نحن أفضل من Respond.io و WATI و SleekFlow؟

| القدرة | محرك النمو الكويتي | المنافسون العالميون |
|---|---|---|
| **ذكاء اصطناعي للهجة الكويتية** | 133+ مؤشر لهجة، مع دعم التبديل اللغوي | لا يوجد دعم للهجات العربية |
| **مدفوعات كي-نت** | تكامل مباشر مع Tap Payments (كي-نت، فيزا، ماستركارد) | Stripe فقط (لا يدعم كي-نت) |
| **شحن محلي** | تكامل مع أرامكس مع تتبع عبر واتساب | لا يوجد تكامل شحن |
| **إقامة البيانات** | AWS البحرين (الخليج)، تصنيف CITRA | استضافة في أمريكا/أوروبا فقط |
| **صفحات هبوط** | منشئ صفحات مدمج مع زر واتساب | غير متاح |
| **استضافة ذاتية** | يمكن التثبيت للبنوك والجهات الحكومية | SaaS فقط |
| **التكلفة** | من 9.900 د.ك/شهر (~32 دولار) | من 99-299 دولار/شهر |

### المنصة في لمحة

- **145 نقطة وصول API** عبر 31 وحدة
- **42 جدول بيانات** مع أمان على مستوى الصفوف PostgreSQL
- **23 صفحة للمستخدم** مع دعم العربية RTL
- **133+ مؤشر للهجة الكويتية** لفهم الذكاء الاصطناعي
- **5 قنوات تواصل** (واتساب، إنستغرام، فيسبوك، سناب شات، دردشة الموقع)
- **9 أنواع عقد للبوت** بما فيها الدفع داخل المحادثة وتتبع الشحن

## 2. البدء والتسجيل

### 2.1 إنشاء حساب جديد

قم بزيارة **app.kwgrowth.com/register** وأدخل:
- اسم الشركة
- الاسم الأول والأخير
- البريد الإلكتروني
- اسم المستخدم
- كلمة المرور (8 أحرف على الأقل)

عند التسجيل، يتم تعيينك تلقائياً بدور **المالك** مع صلاحيات كاملة لجميع ميزات المنصة. يتم إصدار رمز JWT (صالح 15 دقيقة) يتجدد تلقائياً.

### 2.2 متطلبات النظام

| المكون | المتطلب |
|---|---|
| المتصفح | كروم 90+ أو فايرفوكس 88+ أو سفاري 14+ أو إيدج 90+ |
| واتساب | حساب Meta Business مع وصول Cloud API |
| المدفوعات | حساب Tap Payments لقبول كي-نت |
| الجوال | متجاوب بالكامل — يعمل على الهاتف والجهاز اللوحي |

### 2.3 الخطوات الأولى بعد التسجيل

1. **ربط واتساب** — اذهب إلى الإعدادات > القنوات وأدخل بيانات WhatsApp Cloud API
2. **استيراد جهات الاتصال** — ارفع ملف CSV أو أضف جهات اتصال يدوياً
3. **إنشاء أول أتمتة** — أعد رد تلقائي للرسائل الواردة
4. **إعداد خط المبيعات** — أنشئ مراحل البيع وابدأ تتبع الصفقات

## 3. لوحة التحكم الرئيسية

توفر لوحة التحكم نظرة شاملة ومحدثة لحظياً على أعمالك:

### بطاقات المؤشرات الرئيسية (4 مقاييس أساسية)
- **إجمالي جهات الاتصال** — عدد العملاء في نظامك مع مؤشر الاتجاه
- **المحادثات المفتوحة** — المحادثات النشطة عبر واتساب التي تحتاج انتباه
- **الرسائل (الفترة)** — حجم الرسائل الواردة والصادرة
- **الإيرادات (المكسوبة)** — إجمالي قيمة الصفقات بالدينار الكويتي (3 خانات عشرية)

### القائمة الجانبية
تتميز القائمة بتصميم داكن أنيق مع خلفية متدرجة (أزرق داكن)، وأيقونات Lucide SVG احترافية، وشريط أخضر للصفحة النشطة. تتوفر في جميع الصفحات.

### مميزات الرأس العلوي
- **مسار التنقل** — اعرف دائماً أين أنت
- **بحث شامل** — بحث سريع في جهات الاتصال والمحادثات
- **جرس الإشعارات** — تنبيهات فورية للرسائل الجديدة
- **تبديل الوضع الداكن** — التبديل بين المظهر الفاتح والداكن

## 4. إدارة جهات الاتصال والعملاء

### 4.1 قائمة جهات الاتصال (`/contacts`)

تعرض صفحة جهات الاتصال جميع عملائك في جدول بيانات متقدم يدعم:

- **بحث فوري** — اكتب للبحث بالاسم أو الهاتف أو البريد الإلكتروني (مع تأخير 300 مللي ثانية)
- **تصفية متعددة** — فلتر حسب الحالة (نشط/غير نشط/محظور)، المصدر (يدوي/واتساب/استيراد/صفحة هبوط/API)، والوسوم
- **فرز الأعمدة** — انقر على أي عنوان عمود للترتيب تصاعدياً أو تنازلياً
- **إجراءات جماعية** — حدد عدة جهات اتصال للحذف أو إضافة وسم أو تغيير الحالة
- **ترقيم الصفحات** — تنقل بين الصفحات مع 20 جهة اتصال لكل صفحة

### 4.2 صفحة تفاصيل جهة الاتصال (`/contacts/{id}`)

انقر على أي جهة اتصال لعرض ملفها الكامل:
- معلومات الاتصال (الهاتف، البريد، الاسم)
- الحالة والمصدر
- نتيجة العميل المحتمل
- حالة اشتراك واتساب
- الوسوم المعينة (ملونة)
- قيم الحقول المخصصة
- ملاحظات
- وضع التعديل مع الحفظ المباشر

### 4.3 الوسوم (التصنيفات)

الوسوم هي تصنيفات ملونة لتنظيم جهات الاتصال. تُدار من الإعدادات > الوسوم:
- إنشاء وسوم باسم ولون ووصف مخصص
- 10 ألوان مسبقة متاحة
- الوسوم فريدة لكل شركة (بدون تكرار)

### 4.4 الحقول المخصصة

أنشئ حقولاً إضافية لكل جهة اتصال:
- **5 أنواع حقول**: نص، رقم، تاريخ، قائمة اختيارات، نعم/لا
- ترتيب عرض قابل للتعديل
- خيار الحقل الإلزامي

### 4.5 استيراد CSV

ارفع جهات الاتصال بشكل جماعي:
- **تخطيط مرن للأعمدة** — يتعرف تلقائياً على أسماء الأعمدة الشائعة
- **منطق التحديث** — الأرقام المكررة يتم تحديثها، والجديدة يتم إنشاؤها
- **معالجة في الخلفية** — الملفات الكبيرة تُعالج عبر Celery
- **تقارير الأخطاء** — الصفوف الفاشلة تُبلغ مع أرقام السطور

### 4.6 الجدول الزمني الموحد (`/timeline/{contact_id}`)

اعرض جميع التفاعلات مع جهة اتصال عبر جميع القنوات في تدفق زمني واحد:
- رسائل واتساب
- أحداث فتح/إغلاق المحادثات
- أنشطة الصفقات وتغييرات المراحل
- تحديثات تتبع الشحن

**واجهة API:** `GET /v1/timeline/{contact_id}`

## 5. صندوق الوارد الموحد

### 5.1 قائمة المحادثات (`/inbox`)

تعرض اللوحة اليسرى جميع المحادثات مرتبة حسب آخر رسالة:
- **دوائر الصور الرمزية** مع الأحرف الأولى لجهة الاتصال
- **نقطة الحالة** — أخضر (مفتوحة)، أصفر (قيد الانتظار)، أزرق (مؤجلة)، رمادي (مغلقة)
- **شارة غير مقروءة** — عداد أخضر للرسائل غير المقروءة
- **معاينة آخر رسالة** — نص مختصر
- **وقت نسبي** — "منذ 2 دقيقة"، "منذ ساعة"، الخ.
- **بحث وأزرار تصفية** — فلتر حسب الحالة بنقرة واحدة

### 5.2 موضوع الرسائل

تعرض اللوحة اليمنى المحادثة الكاملة:
- **فقاعات دردشة متدرجة** — أخضر للصادرة، أبيض للواردة
- **أيقونات حالة التسليم** — ساعة (قيد الإرسال)، علامة (تم الإرسال)، علامتان (تم الاستلام)، علامتان زرقاء (تمت القراءة)، X (فشل)
- **الطابع الزمني** على كل رسالة
- **تمرير تلقائي** لآخر رسالة
- **Enter للإرسال** (Shift+Enter لسطر جديد)

### 5.3 تحديثات فورية

تتحدث المحادثات تلقائياً كل 3 ثوان. تستخدم المنصة اتصالات WebSocket للتسليم الفوري:

```
wss://app.kwgrowth.com/ws?token=<jwt_access_token>
```

الأحداث المرسلة عبر WebSocket:
- `message.new` — رسالة جديدة واردة/صادرة
- `conversation.updated` — تغيير الحالة أو التعيين
- `message.status` — تحديث حالة التسليم

### 5.4 تكامل WhatsApp Cloud API

تُرسل وتُستقبل الرسائل عبر WhatsApp Cloud API من Meta:

**أنواع الرسائل المدعومة:**
- نص
- صورة، فيديو، صوت، مستند
- قالب (معتمد مسبقاً من Meta)
- تفاعلي (أزرار، قوائم)
- موقع جغرافي
- منتج وقائمة منتجات (كتالوج واتساب)

## 6. الحملات التسويقية والبث الجماعي

### 6.1 إدارة الحملات (`/campaigns`)

أرسل رسائل واتساب جماعية لقاعدة عملائك بالكامل أو لشرائح مستهدفة.

**المميزات:**
- **رسائل قوالب أو نصية** — استخدم قوالب معتمدة من Meta للبث
- **استهداف الجمهور** — جميع جهات الاتصال، حسب الوسم، حسب الشريحة، أو فلتر مخصص
- **جدولة** — أرسل فوراً أو جدول لتاريخ مستقبلي
- **تتبع التسليم** — عدادات فورية للمرسلة والمستلمة والمقروءة والفاشلة
- **إيقاف مؤقت/استئناف** — أوقف حملة قيد الإرسال واستأنفها لاحقاً
- **تحديد المعدل** — يحترم حد واتساب 80 رسالة/ثانية

**نقاط وصول API (8):**
- `GET /v1/campaigns` — قائمة الحملات
- `POST /v1/campaigns` — إنشاء حملة
- `GET /v1/campaigns/{id}` — تفاصيل الحملة
- `PATCH /v1/campaigns/{id}` — تحديث الحملة
- `DELETE /v1/campaigns/{id}` — حذف الحملة
- `POST /v1/campaigns/{id}/send` — بدء الإرسال
- `POST /v1/campaigns/{id}/pause` — إيقاف مؤقت
- `GET /v1/campaigns/{id}/stats` — إحصائيات التسليم

## 7. منشئ بوت المحادثة التفاعلي

### 7.1 قائمة التدفقات (`/chatbots`)

أنشئ تدفقات محادثة آلية باستخدام محرر مرئي بالعقد.

**أنواع المحفزات:**
- **مطابقة الكلمات المفتاحية** — يتفعل عندما تحتوي الرسالة على كلمات محددة
- **أي رسالة واردة** — يتفعل مع كل رسالة جديدة
- **محادثة جديدة** — يتفعل عند بدء محادثة لأول مرة
- **Webhook خارجي** — يتفعل عبر استدعاء من نظام خارجي

### 7.2 محرر التدفق (`/chatbots/{id}`)

أنشئ التدفقات بإضافة وربط العقد:

| نوع العقدة | الوصف | حالة الاستخدام |
|---|---|---|
| **إرسال رسالة** | إرسال نص للعميل | رسالة ترحيب، رد معلومات |
| **طرح سؤال** | سؤال وانتظار الإجابة | تأهيل العميل، جمع بيانات |
| **شرط** | تفريع حسب المحتوى | توجيه حسب اللغة أو النية |
| **تأخير** | انتظار عدد محدد من الثواني | متابعة مؤقتة |
| **تعيين موظف** | تحويل لموظف بشري | استفسارات معقدة |
| **إجراء** | وسم، تحديث حقل، نقاط | أتمتة CRM |
| **رابط دفع** | إنشاء رابط Tap Payment | شراء داخل المحادثة |
| **تتبع الشحن** | استعلام عن حالة الشحنة | "وين طلبي؟" |
| **استدعاء API** | طلب HTTP خارجي | تكامل مع طرف ثالث |

### 7.3 عقدة رابط الدفع

تُنشئ عقدة رابط الدفع شحنة عبر Tap Payments وترسل رابط الدفع مباشرة في محادثة واتساب:

```json
{
  "type": "payment_link",
  "data": {
    "amount": 29.900,
    "currency": "KWD",
    "description": "اشتراك شهري"
  }
}
```

يستلم العميل: "الرجاء إكمال الدفع بقيمة 29.900 د.ك لـ: اشتراك شهري. ادفع هنا: https://tap.company/pay/..."

### 7.4 عقدة تتبع الشحن

عندما يسأل العميل "وين طلبي؟"، تقوم عقدة تتبع الشحن تلقائياً بـ:
1. البحث عن أحدث شحنة للعميل
2. استرجاع حالة التتبع من شركة الشحن
3. إرسال رسالة حالة منسقة مع رقم التتبع والناقل وموعد التسليم المتوقع

- الحالة (نجاح/فشل)
- عدد الإجراءات المنفذة
- المدة بالمللي ثانية
- تفاصيل الخطأ في حالة الفشل

## 9. محرك الذكاء الاصطناعي للهجة الكويتية

### 9.1 نظرة عامة

محرك الذكاء الاصطناعي هو أقوى ميزة تنافسية للمنصة. لا يوجد منافس يقدم فهماً أصلياً للهجة الكويتية العربية.

### 9.2 اكتشاف اللهجة

**133+ مؤشر للهجة الكويتية** مُنظمة في 9 فئات:
- التحيات (شلونك، هلا والله، يا بعد قلبي)
- الضمائر (شنو، ليش، وين، هالحين)
- المكثفات (وايد، مرة، خلاص، بس)
- الأفعال (أبي، أسوي، أدري، خلنا)
- الأسماء (فلوس، هدوم، سيارة، ديوانية)
- التعبيرات (الله يبارك، ما عليك، عسى بس)
- مصطلحات تجارية (كم السعر، خصم، عرض، توصيل)

**فئات اللهجات المكتشفة:** كويتي، خليجي، فصحى، إنجليزي، مختلط

### 9.3 دعم التبديل اللغوي (Code-Switching)

يفهم المحرك الرسائل التي تخلط بين العربية والإنجليزية — وهو شائع جداً في الكويت:

مثال: "hala shlonik, aby delivery for the new offer please"
- اكتشاف: لهجة `كويتية`
- ترجمة حرفية: `shlonik` ← شلونك، `aby` ← أبي
- النية: `شراء`

**27 تعبير مكتوب بالحروف اللاتينية** بما فيها: shlonik، inshallah، mashallah، yallah، khalas، habibi، zain، wallah، وغيرها.

### 9.4 تصنيف النوايا (10 فئات)

كل رسالة واردة تُصنف في واحدة من 10 نوايا مع نسب ثقة:

| النية | كلمات عربية | كلمات إنجليزية |
|---|---|---|
| تسعير | كم، سعر، بكم، خصم | price, cost, discount, offer |
| شراء | أبي، شراء، اشتري، أطلب | buy, order, want, checkout |
| دعم | مساعدة، مشكلة، خربان | help, problem, issue, broken |
| تحية | هلا، شلون، مرحبا | hi, hello, hey |
| استفسار | أبي أعرف، استفسار، تفاصيل | info, details, tell me |
| جدولة | موعد، حجز، متى | appointment, book, available |
| شحن | وين طلبي، شحن، تتبع | tracking, delivery, where is my order |
| شكوى | شكوى، زعلان، سيء | complaint, angry, terrible |
| تقييم | شكرا، ممتاز، رائع | thanks, great, excellent |
| إلغاء | إلغاء، رجعوا فلوسي | cancel, refund, unsubscribe |

### 9.5 تحليل المشاعر

كل رسالة تحصل على درجة مشاعر:
- **إيجابي** (0 إلى +1.0): 36 كلمة مؤشرة (19 عربية، 17 إنجليزية)
- **سلبي** (-1.0 إلى 0): 33 كلمة مؤشرة (18 عربية، 15 إنجليزية)
- **محايد** (0.0)
- **مختلط** (كلمات إيجابية وسلبية معاً)

### 9.6 قوالب الرد التلقائي

عند اكتشاف اللهجة الكويتية، يقترح الذكاء الاصطناعي ردوداً مناسبة ثقافياً:

| النية | قالب الرد |
|---|---|
| تحية | هلا والله! شلون أقدر أساعدك اليوم؟ |
| تسعير | تبي تعرف الأسعار؟ خلني أوريك التفاصيل |
| شراء | ممتاز! تبي أسويلك الطلب الحين؟ |
| دعم | ما عليك هم، خلني أساعدك بالمشكلة |
| شحن | خلني أشيك على الطلب حقك |

### 9.7 تكامل Claude API

عند تكوين مفتاح Anthropic API، يستخدم المحرك **Claude Haiku** للتحليل المتقدم:
- فهم كامل على مستوى الجملة
- تصنيف سياقي للنوايا
- توليد ردود طبيعية بلهجة العميل
- تلخيص المحادثات
- استخراج رؤى العميل (الاحتياجات، الإلحاح، تفضيل اللغة)

**نقاط وصول API:**
- `POST /v1/ai/analyze` — تحليل رسالة
- `GET /v1/ai/context/{conversation_id}` — الحصول على سياق AI للمحادثة
- `POST /v1/ai/suggest/{conversation_id}` — توليد رد مقترح

## 18. الامتثال وأمن البيانات

### 18.1 تصنيف البيانات حسب CITRA

تُصنف البيانات وفقاً لأنظمة هيئة الاتصالات وتقنية المعلومات الكويتية (CITRA):

| المستوى | التصنيف | متطلبات الاستضافة | بيانات المنصة |
|---|---|---|---|
| 1 | بيانات عامة | أي منطقة | صفحات الهبوط، كتالوج المنتجات |
| 2 | بيانات داخلية | يُفضل الخليج | التحليلات، سجلات التدقيق، الصفقات |
| 3 | بيانات سرية | الكويت/CITRA الخليج مطلوب | جهات الاتصال، المحادثات، المدفوعات، سياقات AI |
| 4 | بيانات حكومية/مقيدة | الكويت فقط | بيانات الجهات الحكومية (يتطلب مركز بيانات داخل الكويت) |

### 18.2 قائمة فحص الامتثال (8 نقاط)

1. **إقامة البيانات** — AWS me-south-1 (البحرين، الخليج)
2. **التشفير أثناء التخزين** — تشفير قرص PostgreSQL بمعيار AES-256
3. **التشفير أثناء النقل** — TLS 1.3 لجميع الاتصالات
4. **تتبع الموافقة** — حالة اشتراك واتساب لكل جهة اتصال
5. **سياسة الاحتفاظ بالبيانات** — قابلة للتكوين (الافتراضي 365 يوم)
6. **سجل التدقيق** — تسجيل جميع الإجراءات مع التاريخ والمستخدم والتغييرات
7. **حق الحذف** — حذف مرن مع إمكانية الحذف النهائي
8. **تصدير البيانات** — تصدير CSV لجميع أنواع البيانات

### 18.3 إجراءات الأمان

- **المصادقة**: JWT (HS256)، رموز وصول 15 دقيقة، تجديد 7 أيام
- **كلمات المرور**: bcrypt مع 12 جولة
- **العزل المتعدد**: أمان على مستوى الصفوف PostgreSQL (عزل company_id)
- **نظام الأدوار**: 5 أدوار، 34 صلاحية، فرض لكل نقطة وصول
- **تحديد المعدل**: نافذة منزلقة Redis (30/ث API، 5/ث مصادقة، 100/ث webhooks)
- **التشفير**: Fernet (AES-128-CBC) لمفاتيح API والرموز المخزنة

## 20. الخاتمة

يقدم **محرك نمو واتساب الكويت** ما لا يستطيع أي نظام CRM عالمي تقديمه: حل مؤسسي شامل صُمم من الأساس للسوق الكويتي.

### للشركات

- **تقليل وقت الاستجابة** مع ردود تلقائية بالذكاء الاصطناعي باللهجة الكويتية
- **زيادة معدلات التحويل** مع الدفع داخل المحادثة وكتالوجات المنتجات
- **تبسيط العمليات** مع سير عمل مؤتمت وتوجيه ذكي للموظفين
- **تتبع كل شيء** مع تحليلات فورية عبر جهات الاتصال والصفقات والحملات

### للشركاء

- **جاهز للعلامة البيضاء** — خيار استضافة ذاتية للبنوك والحكومة
- **بنية API أولاً** — 145 نقطة وصول للتكامل السلس
- **متعدد المستأجرين** — خدمة عملاء غير محدودين من تثبيت واحد
- **متوافق مع الخليج** — تصنيف بيانات CITRA، استضافة البحرين، سجلات التدقيق
- **ميزة التكلفة** — أرخص بنسبة 60-70% من Respond.io مع مميزات كويتية أكثر

### الأساس التقني

- **الخلفية**: Python 3.11، FastAPI، SQLAlchemy 2.0 (غير متزامن)، Celery، Redis
- **الواجهة**: Next.js 15، TypeScript، Tailwind CSS v4، أيقونات Lucide
- **قاعدة البيانات**: PostgreSQL 16 مع أمان على مستوى الصفوف
- **البنية التحتية**: Docker، Terraform (AWS)، GitHub Actions CI/CD
- **الذكاء الاصطناعي**: Claude Haiku + محرك قواعد 133 مؤشر
- **المدفوعات**: Tap Payments (كي-نت، فيزا، ماستركارد)
- **الشحن**: أرامكس الكويت مع تتبع واتساب
